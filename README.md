# 📊 Vendor Performance Analytics

This repository contains an end-to-end analytics project focused on vendor performance, covering sales, purchasing, inventory, and profitability. The work combines SQL for data extraction, Python for data preparation, and Power BI for interactive reporting and business insights.

---

## 📌 Project Overview

Vendor performance is a critical factor in supply chain efficiency and profitability. By evaluating supplier behavior, procurement costs, inventory movement, and revenue contribution, organizations can make more informed sourcing and operational decisions.

This project transforms raw transactional data into structured analysis and dashboards that support better business decision-making.

---

## 🎯 Business Problem

Retail and wholesale organizations generate large volumes of sales, purchase, and inventory data, making it difficult to accurately evaluate vendor performance and profitability. Without a structured analytical approach, businesses may experience reduced profit margins, inefficient inventory management, and suboptimal procurement decisions.

This project aims to analyze vendor performance using SQL, Python, and Power BI to identify opportunities for improving operational efficiency and profitability through data-driven insights.

---

## 🎯 Project Objectives

The project focuses on answering key business questions, including:

- Identify underperforming brands requiring pricing or promotional adjustments.
- Determine vendors contributing the highest sales and gross profit.
- Analyze the effect of bulk purchasing on unit procurement costs.
- Evaluate inventory turnover to reduce holding costs.
- Compare profitability between high-performing and low-performing vendors.
- Discover sales and purchasing trends that support strategic decision-making.
- Build an interactive dashboard for business users and decision-makers.
